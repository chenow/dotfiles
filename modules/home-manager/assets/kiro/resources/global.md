# Kiro CLI Global Context

## Documentation & Research

1. Fetch latest documentation using Context7 when available
1. Find examples using GitHub MCP server inside repositories with significant stars and recent activity
1. Always return a list of links used to generate the response at the end

## Tool Priority

Always use tools in this order:

1. Context7 MCP for documentation
1. GitHub MCP for code examples
1. web_fetch and web_search as last resort
